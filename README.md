# 🚀 CodeFix AI

An AI-powered code debugger that detects errors, suggests fixes, and explains code using LLM.

---

## 🔥 Overview

CodeFix AI is a full-stack web application that helps developers and beginners debug code instantly.  
Users can paste their code, and the system analyzes it using AI to identify errors, generate corrected code, and provide clear explanations.

---

## ✨ Features

- 🐞 Detect code errors automatically  
- 🛠️ Generate corrected code  
- 🧠 Provide human-readable explanations  
- ⚡ Real-time debugging interface  
- 💻 Clean and responsive UI  

---

## 🧑‍💻 Tech Stack

| Layer        | Technology |
|-------------|-----------|
| Frontend     | React.js (Vite) |
| Backend      | Node.js, Express.js |
| AI Engine    | Groq LLM API |
| HTTP Client  | Axios |
| Config       | dotenv |
